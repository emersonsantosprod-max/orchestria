Invoke the brainstorming skill before any implementation.

Load `.claude/skills/brainstorming/SKILL.md` and follow its flow:
1. Explore context (ask one clarifying question at a time if needed)
2. Present 2–3 approaches with trade-offs
3. Produce a design spec in `docs/specs/YYYY-MM-DD-<topico>.md`
4. Wait for explicit approval before writing any code

Gate: do not write production code until the user approves the design.
