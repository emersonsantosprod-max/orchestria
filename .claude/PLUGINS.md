# Plugins Roadmap

Candidate plugins for future installation. Not installed yet.

| Plugin | Purpose | Priority |
|--------|---------|----------|
| pytest MCP | Run tests, inspect failures from Claude | High |
| SQLite MCP | Browse `automacao.db` read-only from Claude | Medium |
| FastAPI codegen | Generate route stubs and schemas from domain types | Low (API not yet started) |

When ready to install, run:
```bash
claude mcp add <plugin-name>
```
